What is VideoMCC?

Video Multiple Choice Caption (VideoMCC) is a new task for video understanding and a new benchmark for this task. We believe that VideoMCC is a good option for developing and evaluating algorithms for video understanding at a high-semantic level.

VideoMCC problem statement

Given a video clip and a set of 5 English sentences, the task is to determine which sentence best describes the visual content of the input clip.
